# Dice Simulator

## Introduction
Dice Simulator is a simple command-line game that simulates the rolling of a standard six-sided dice. This Python-based simulator allows users to roll a dice, view the result, and keep track of their total score across multiple rounds.

## Features
- **Simple and Intuitive**: Easy-to-use command-line interface.
- **Score Tracking**: Keeps track of the total score and number of rounds played.
- **Repeat Play**: Allows users to play multiple rounds until they decide to stop.

## Prerequisites
Before you run this game, make sure you have Python installed on your computer. 

## Installation
No additional installation is required. Just make sure Python is installed on your system.

## How to Play
- After running the script, the game will automatically roll a dice.
- The dice result will be shown as a graphical representation in the command-line interface.
- After each roll, you will be prompted to play again. Type `y` to continue or `n` to stop playing.
- If you choose to stop playing, the game will display the total number of rounds played and your total score before exiting.
