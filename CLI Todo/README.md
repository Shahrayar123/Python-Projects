<h1>Todo app</h1>

* How Does It Works

<p>◾To add an Item:</p>
<code >
Add NameOfTheTodo
</code>

<br>
<p>◾To See The Todos:</p>
<code >
Show
</code>

<br>
<p>◾To Edit a Todos:</p>
<code >
Edit NumberOfTheTodo
</code>

<br>
<p>◾To Delete Or Complete A Todo:</p>
<code >
Complete NumberOfTheTodo
</code>

<br>
<p>◾To Exit Todo:</p>
<code >
Exit 
</code>
<br>
<p>Or</p>
<code>
Ctrl + d
</code>
